# 🏗️ Infra-lab-homelab

Este repositorio documenta el diseño, implementación y evolución de mi laboratorio de infraestructura personal. El objetivo principal es desarrollar habilidades en administración de sistemas Linux, orquestación de contenedores y networking avanzado.

## 🛠️ Stack Tecnológico
- **OS:** Ubuntu Server 22.04 LTS
- **Virtualización:** Docker & Docker Compose
- **Networking:** Nginx (Proxy Inverso), DNS local y segmentación de redes.
- **Cloud:** Google Cloud Platform (GCP) para integración híbrida.

## 🗺️ Roadmap de Aprendizaje
- [x] Configuración inicial de Ubuntu Server y Docker.
- [ ] **Nivel 1:** Implementación de Nginx como Reverse Proxy para múltiples servicios.
- [ ] **Nivel 2:** Seguridad de red (Firewalls, redes internas de Docker).
- [ ] **Nivel 3:** Observabilidad y Monitoreo con Grafana + Prometheus.

## 📂 Organización del Proyecto
- **/networking**: Archivos de configuración de Nginx y protocolos de red.
- **/services**: Despliegue de aplicaciones mediante Docker Compose.
- **/scripts**: Automatización de tareas de sistema.
